## 캡처된 화면 단계

각 프로필마다 다음 10개 화면이 캡처되었습니다:

1. **01-main.png**: 메인 페이지 (검사 시작 전)
2. **02-basic-info-empty.png**: 기본정보 입력 페이지 (빈 화면)
3. **03-basic-info-filled.png**: 기본정보 입력 완료
4. **04-consultation-empty.png**: 상담내용 입력 페이지 (빈 화면)
5. **05-consultation-filled.png**: 상담내용 입력 완료
6. **06-assessment-q1.png**: 성향테스트 첫 번째 질문
7. **07-assessment-q8.png**: 성향테스트 중간 질문 (8번)
8. **08-assessment-last.png**: 성향테스트 마지막 질문 (24번)
9. **09-results-initial.png**: 결과 페이지 초기 화면
10. **10-results-full.png**: AI 분석 완료 후 전체 결과

## 테스트 구성

### 페이지 플로우
```
메인 페이지 (/)
  ↓
기본정보 입력 (/basic-info)
  ↓
상담내용 입력 (/consultation)
  ↓
성향테스트 (/assessment) - 24개 질문
  ↓
결과 페이지 (/results) - AI 분석 결과
```

### 입력 데이터 구성

#### 기본정보
- 애칭 (nickname)
- 나이 (age)
- 성별 (gender)
- 활동 (activities): 학원, 방과후, 운동
- 취미 (hobbies): 여가 활동
- 관심사 (interests): 주제별 관심
- 잘하는 과목 (strongSubjects)
- 받은 상 (achievements)

#### 상담내용
- 학교 피드백 (schoolFeedback)
- 학원 피드백 (academyFeedback)

#### 성향테스트
- 24개 질문 × 5점 척도
- 카테고리: 창의·예술형, 분석·연구형, 사람·돌봄형, 리더·조직형, 실무·기술형

## 기술 스택
- **테스트 프레임워크**: Playwright
- **브라우저**: Chromium
- **실행 모드**: Headless

## 파일 위치
- 테스트 파일: `tests/child-profiles.spec.ts`
- 설정 파일: `playwright.config.ts`
- 스크린샷: `tests/screenshots/{프로필명}/`
- 테스트 리포트: `playwright-report/`

## 실행 방법
```bash
# 모든 테스트 실행
npx playwright test

# 특정 프로필만 테스트
npx playwright test --grep "창의적-예술형"

# UI 모드로 실행
npx playwright test --headed

# 테스트 리포트 보기
npx playwright show-report
```
